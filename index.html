Alguns dos meus projetos

Conecta Uerj

Atuo no Conecta Uerj como desenvolvedor Java backend utilizando banco de dados Postgres. Esta plataforma robusta é um projeto de prodocência da Uerj para o gerenciamento de eventos, palestras, projetos, etc, da universidade. O sistema atua desde a inscrição do evento até a emissão e registro de certificados, além disso, possui um portal para divulgação de novidades da faculdade e registra as atividades complementares dos alunos.

Facilita Uerj

Atuo desde a modelagem do sistema e elaboração de regras de negócio, como também desenvolvedor fullstack com Java, Angular e Mysql. O Facilita Uerj é um projeto de desenvolivemnto voluntário para melhorar a comunição entre Alunos e Coodenadores. O sistema fornece funcionalidades para o agendamento de reuniões através de uma interface elegante, além disso o sistema possui integração com a Api do Google Calendar para facilitar a organização dos coordenadores.

Investor Bee

Esta aplicação backend é um agregador de investimento para o gerenciamento de ações diretamente da bolsa de valores, nesta aplicação consumo a API da Brabi, via OpenFeign, para rebecer dados dos valores das ações, e forneço cruds para o gerenciamento da carteira de investimentos do usuário. Nesta aplição, também implementei a conteinerização com Docker, toda padronização de Api Restfull, serviço de segurança com JWT, documentação via Swagger, testes unitários com Junit e Mockito e persistência de dados no MySQL.

API para Controle de Produtos

Esta aplicação fornece uma API para o cadastro de produtos em uma loja. Além de permitir o registro de produtos e seus respectivos fornecedores, a aplicação também envia um e-mail para o fornecedor notificando-o sobre a chegada do produto na loja. Utilizo toda padronização para criação de Api Rest, com a implementação de autenticação via JWT Token, envio de emails personalizados com Thylemeaf e JavaEmailSender, além de estar documentar com Swagger e persistir dados no banco de dados MySql.

Pesquisador de Palavras em Arquivos de Texto com Árvore AVL

Este projeto implementa uma árvore AVL em Java para pesquisa de palavras em coleções de arquivos de texto. Facilita a busca por palavras específicas, fornecendo informações sobre sua frequência e ocorrência.

API para Controle de Medicamentos

Esta aplicação fornece uma API para o gerenciamento de remédios. Neste projeto implemento mecanismos de segurança via JWT Token, CRUD e persistência de dados com JPA/Hibernate e MySQL.

Atualmente, estou em busca de uma oportunidade de estágio em ambas as áreas, para que eu possa continuar aprendendo e aprimorando meus conhecimentos no campo da tecnologia.
